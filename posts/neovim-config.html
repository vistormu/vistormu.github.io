How to Set Up Neovim from Scratch

Published on oct 15th 2023. Last edit on oct 15th 2023.

Neovim is the superior technology for text editing. You do not need IDEs or waiting for slow apps to open. Editing in the terminal is just enough. Neovim is super fast, lightweight and highly costumizable. You will learn how to adapt Neovim to your preferences, achieving high editing speeds. Moreover, once you get used to Vim Motions there will be no way back.

A very popular Neovim configuration guide is kickstart.nvim, which is also a very good way to start the Neovim journey. Also, I highly recommend ThePrimeagen's video on the topic, which is a thirty-minute video that covers everything you need to know. However, if you don't watch it at 0.5x speed you could die of anxiety.

If you don't feel like configuring your own Neovim, there are many pre-configured Neovim distributions out there. The most popular ones are NvChad and LunarVim. However, using these bad bois will result in learning the key-bindings of others and not yours.

This blog post is a guide on my minimal Neovim configuration which is highly based on the ThePrimeagen's video I mentioned before. I will also cover necessary installations in your system that are not mentioned in the video.

Lastly, if you are not using a Linux-based operating system this blog post is not for you. Sorry not sorry.

Installing Neovim

The first step, of course, is to download and install Neovim. If you are using a Debian-based Linux distribution, you can install Neovim with the following command:

terminal
sudo snap install --edge nvim --classic

If you are using an Arch-based Linux distribution, you can install Neovim with the following command:

terminal
sudo pacman -S neovim

If you are using a different Linux distribution, like Fedora, you can install Neovim with the following command:

terminal
sudo dnf install neovim

It is very important to download a version of Neovim higher that 0.9.0, as many plugins are not supported for lower versions.

If everything went well, you should be able to run Neovim with the following command:

terminal
nvim
