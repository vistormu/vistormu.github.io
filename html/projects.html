projects

cahier
#rust #ssh #cli

Cahier is a tool that allows you to manage your SSH connections in a simple and efficient way. It supports sending SSH keys, bringing and sending files, and connecting to your servers with a single command.

VISIT
proxima
#go #web-development

Proxima is a markup language that transpiles Proxima source code into HTML. It offers clearer syntax instead of plain HTML/CSS coding. It is a fun project that I started to learn more about the Go programming language, and it ended up being a useful tool for my personal website.

VISIT
catasta
#python #pytorch #machine-learning

Catasta is a Python library designed to simplify the process of Machine Learning model experimentation. Optimization, training, evaluation and inference all in one place! It is a part of my research work and it is available on PyPI.

VISIT
vistormu.dev
#proxima #html #css #go

This is my personal website, where my projects, blog posts, research work, and other stuff are hosted. It is built using Proxima, a markup language that I created to simplify the process of writing HTML and CSS code.

VISIT
vclog
#python #logging

Vistor's Console Logger (VCLog) offers a minimal and colorful console logging. Single-script. No dependencies. It is available on PyPI.

side projects

snake in the CLI
#go #cli

This is my attempt to create a simple Snake game using only the terminal and Go. It is a fun project to learn more about the terminal and its capabilities. I have a a blog post on how to do it.

VISIT
jamba
#go #text-editor

Jamba is a text editor inspired by Vim and written in Go. It is based on the antirez's Kilo Text Editor, and it is my attempt to port the C version to a more idiomatic Go code. I highly recommend you to try and build your own text editor with this wonderful tutorial!

VISIT
kimchi
#go #interpreter #programming-language

Kimchi is a general-purpose programming language that was part of a personal project to learn about programming language design and implementation. It is written in Go and it is based on the book Writing an Interpreter in Go by Thorsten Ball.

VISIT

Have you ever felt like you don't want to decide between a series of options? Decision Maker helps your indecision by running a board game!
