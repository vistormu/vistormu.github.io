Jamba
#go

Jamba is a superfast and lightweight text editor inspired by Vim and written in Go. It is based on the antirez's Kilo Text Editor, and it is my attempt to port the C version to a more idiomatic Go code. Also, I extended it to resemble Vim, which has been so fun. I highly recommend you to try and build your own text editor with this wonderful tutorial!

VISIT
Kimchi
#go

Kimchi is a general-purpose programming language that aims to have a simple, consistent and coherent syntax. Kimchi was part of a personal project to learn about programming language design and implementation. It is written in Go and it is based on the book Writing an Interpreter in Go by Thorsten Ball.
